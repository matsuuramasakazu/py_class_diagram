# py_class_diagram

Pythonで作成されたシンプルなUMLクラス図作図ツールです。直感的なマウス操作でクラス図を作成し、Mermaid形式での保存・読み込みが可能です。

## 主な機能

- **クラスの作成と編集**: クラスの名前、属性、操作を個別に追加・編集できます。
- **多彩なリレーションシップ**: 以下の6種類の関係を描画できます。
  - 継承 (Generalization)
  - 実現 (Realization)
  - 合成 (Composition)
  - 集約 (Aggregation)
  - 依存 (Dependency)
  - 関連 (Association)
- **直感的なキャンバス操作**: 
  - ドラッグ＆ドロップによるクラスの移動
  - ラバーバンド（範囲指定）による複数選択
- **インライン編集**: 各クラスのセクションをダブルクリックすることで、その場でテキストを編集できます。
- **保存と読み込み**: 
  - 図の構造と配置情報を単一の **Markdown形式 (`.md`)** で保存。
  - GitHubやVS Codeなどの標準的なMarkdownビューアでクラス図をプレビュー可能。

## 操作方法

### ツールバー
- **Save**: 現在の図面を保存します。
- **Load**: 保存された図面を読み込みます。
- **Add Class**: キャンバスに新しいクラスを追加します。
- **Select**: クラスの選択・移動を行う基本モードです。
- **各リレーションシップボタン**: 特定の関係を作成するモードに切り替えます。

### キャンバス操作
- **選択と移動**: `Select` モードでクラスをクリックして選択、ドラッグして移動します。
- **範囲選択**: 何もない場所からドラッグすると、枠内のクラスをまとめて選択できます。
- **関係の作成**:
  1. ツールバーで作成したい関係（例: `Generalization`）を選択します。
  2. 接続元のクラス上でマウスを押し下げ、接続先のクラスまでドラッグして放します。
- **インライン編集（ダブルクリック）**:
  - **クラス名**: クラス上部のヘッダーをダブルクリック。
  - **属性**: クラス中央部をダブルクリック。1行に1つの属性を入力します。
  - **操作**: クラス下部をダブルクリック。1行に1つのメソッドを入力します。
  - *編集の確定*: 入力エリア外をクリックするか、`Enter` キー（名前のみ）を押します。

### キーボードショートカット
- `Delete`: 選択中のクラス（および関連する接続線）を削除します。
- `Ctrl + S`: 図面を保存します。

## データ保存について
本ツールは保存時に単一の **`.md`** ファイル（Markdown）を出力します。このファイルは以下の3つのセクションで構成されています。

1. **Header**: 図のタイトルや説明（通常のテキスト）。
2. **Visual Section**: Mermaid形式のクラス図定義（` ```mermaid ` ブロック）。GitHub等でのプレビューに使用されます。
3. **Data Section**: HTMLコメント内に隠されたJSONデータ。クラスの座標やサイズなどのエディタ専用メタデータを保持します。

この形式により、特別なツールがなくても設計内容をGitのdiffや各種Markdownビューアで簡単に確認・共有できます。
